## Color Callouts

<Note type="info" title="Info note">
Positive and neutral messaging renders with the current accent color.
</Note>

<Note type="warning" title="Warning note">
Warnings lean on amber hues that remain legible on dark backgrounds.
</Note>

<Note type="danger" title="Error note">
Error states use rose tones with sufficient contrast against the card background.
</Note>

### Multi-language Group

<CodeGroup title="Initialize client">

```ts
import { createClient } from '@example/sdk'

const client = createClient({
  apiKey: process.env.EXAMPLE_TOKEN!,
})

await client.messages.send({
  to: '+15551234567',
  template: 'otp',
  data: { code: '123456' },
})
```

```python
from example import Client

client = Client(api_key=os.environ["EXAMPLE_TOKEN"])

client.messages.send(
    to="+15551234567",
    template="otp",
    data={"code": "123456"},
)
```

</CodeGroup>
